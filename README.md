Telegram-бот для учета и управления кальянами в кальянной во время рабочей смены. Бот автоматически записывает время добавления каждого кальяна и позволяет вести полный учет по типам и столам.

> ⚡ **Быстро начать?** → Смотрите [Quick Start Guide](QUICKSTART.md)

## ✨ Основные функции

- ✅ **Открытие/закрытие смены** — управление рабочей сменой с автоматической фиксацией времени
- ✅ **Добавление кальянов** — с автоматической записью времени, выбором типа и стола
- ✅ **Просмотр и редактирование** — изменение типа кальяна или перемещение на другой стол
- ✅ **Удаление кальянов** — удаление ошибочно добавленных кальянов
- ✅ **История смен** — просмотр предыдущих смен с детальной информацией
- ✅ **Inline-кнопки** — удобное управление прямо из чата
- ✅ **SQLite БД** — все данные хранятся локально

## 🎯 Типы кальянов

- **Стандарт** — обычный кальян
- **Сигара** — кальян премиум класса

## 📍 Доступные столы

- Столы: **1, 2, 3, 4, 5, 6, 7**
- **Большой бар** — барная стойка большая
- **Малый бар** — барная стойка малая  
- **Стол у танцпола** — специальный стол возле танцпола

## 🚀 Установка

> 💡 **Для быстрого старта в 5 минут смотрите [Quick Start Guide →](QUICKSTART.md)**

### Требования
- **Python 3.10+**
- **pip** или **poetry**

### Шаги установки

1. **Клонируйте репозиторий:**
   ```bash
   git clone https://github.com/username/schetchik-kalikov.git
   cd schetchik-kalikov
   ```

2. **Создайте виртуальное окружение (рекомендуется):**
   ```bash
   # Windows
   python -m venv venv
   venv\Scripts\activate

   # macOS / Linux
   python3 -m venv venv
   source venv/bin/activate
   ```

3. **Установите зависимости:**
   ```bash
   pip install -r requirements.txt
   ```

4. **Получите токен бота:**
   - Откройте Telegram и найдите бота **@BotFather**
   - Отправьте команду `/newbot`
   - Следуйте инструкциям
   - Скопируйте полученный токен

5. **Настройте конфигурацию:**
   ```bash
   # Скопируйте шаблон
   cp .env.example .env
   ```
   
6. **Добавьте токен в `.env`:**
   ```env
   BOT_TOKEN=123456789:ABCdefGHIjklmnoPQRstuvWXYZabcdefg
   ```

## 📖 Использование

### Основной цикл работы

1. **Начало смены:**
   - Отправьте боту `/start`
   - Нажмите **"Открыть смену"**

2. **Добавление кальянов:**
   - Нажмите **"Добавить кальян"**
   - Выберите тип: **Стандарт** или **Сигара**
   - Выберите стол
   - ✅ Кальян добавлен с автоматической фиксацией времени

3. **Просмотр кальянов:**
   - Нажмите **"Текущие кальяны"**
   - Выберите кальян из списка для редактирования

4. **Редактирование кальяна:**
   - Нажмите на кальян
   - Выберите: изменить тип, изменить стол или удалить

5. **Закрытие смены:**
   - Нажмите **"Закрыть смену"**
   - Подтвердите закрытие
   - 📊 Смена закрыта с подсчетом всех кальянов

6. **Просмотр истории:**
   - Нажмите **"История смен"**
   - Выберите смену для просмотра деталей

## 💾 База данных

Бот использует **SQLite** для хранения данных. База данных создается автоматически при первом запуске.

### Таблицы:

**shifts** — смены
- `id` — уникальный ID смены
- `open_time` — время открытия смены
- `close_time` — время закрытия смены
- `is_open` — статус смены (1 = открыта, 0 = закрыта)
- `total_hookahs` — общее количество кальянов за смену

**hookahs** — кальяны
- `id` — уникальный ID кальяна
- `shift_id` — ID смены, к которой относится кальян
- `hookah_type` — тип кальяна (Стандарт/Сигара)
- `table_name` — название стола
- `created_at` — время добавления кальяна

## 📋 API Boта - Inline Кнопки

### Главное меню
- `open_shift` — открыть смену
- `close_shift` — закрыть смену
- `add_hookah` — добавить кальян
- `current_hookahs` — просмотреть текущие кальяны
- `history` — история смен

### Добавление кальяна (FSM States)
- `type_Стандарт`, `type_Сигара` — выбор типа
- `table_[название]` — выбор стола

### Редактирование кальяна
- `view_[id]` — просмотр кальяна
- `edit_type_[id]` — редактировать тип
- `edit_table_[id]` — редактировать стол
- `delete_[id]` — удалить кальян

## 🐛 Решение проблем

### Бот не запускается
```
❌ Токен бота не установлен!
```
**Решение:** Проверьте, что в `.env` файле указан корректный `BOT_TOKEN`

### Ошибка подключения к БД
```
❌ database.sqlite3: no such table
```
**Решение:** Удалите `hookah_bot.db` - БД будет пересоздана автоматически

### Бот не отвечает на команды
**Решение:** 
- Убедитесь, что бот запущен
- Проверьте интернет соединение
- Перезагрузите бот: `Ctrl+C` и снова `python bot.py`
