# 🍃 Счетчик Кальянов — Telegram бот

Телеграм-бот для учета кальянов в кальянной: управление сменами, заявки на кальяны, статусы, уведомления и профиль с ролью.

## ✨ Что умеет бот

### Управление сменами
- Открывать и закрывать смены
- **Редактировать время открытия/закрытия смены** (админ-панель)
- **Переоткрывать закрытые смены** (админ-панель)
- Удалять смены

### Управление кальянами
- Добавлять кальяны с выбором типа, силы, холодности и комментариев
- Принимать кальяны мастером и переводить их в статус готовности
- **Менеджер может редактировать:** тип, стол, силу, холодность и комментарий кальяна
- Удалять кальяны
- Отображать список текущих кальянов и историю смен

### Функциональность
- Глобальные роли пользователей (применяются ко ВСЕМ сменам)
- Уведомлять участников о новых кальянах
- Рассылать уведомление о принятом кальяне в отчётный чат при наличии `REPORT_CHAT_ID`
- Уведомлять всех участников смены и подписанных пользователей о готовых кальянах
- Хранить данные в SQLite
- Показывать профиль пользователя с ролью и статистикой
- Работать в московском времени (UTC+3)

## 🧩 Роли

Роли определяются **глобально** и применяются ко **всем сменам**:

- `admin` — полный доступ к админской панели, управлению сменами и пользователями
- `manager` — **редактирование кальянов** (тип, стол, сила, холодность, комментарий)
- `hookah_master` — добавление и принятие кальянов в работу
- `supervisor` — наблюдение за сменой и отчетность
- `member` — участие в смене и просмотр данных

### Как назначить глобальную роль
1. Админ-панель → 🔧 **Роли**
2. Выбрать пользователя
3. Выбрать роль
4. Роль применяется ко ВСЕМ сменам автоматически

## 📌 Профиль пользователя

В профиле отображается:

- Статус роли
- Имя пользователя из базы
- Статистика кальянов для кальянного мастера:
  - сколько кальянов забито в твоих сменах
  - количество стандартных и сигар
  - общее число кальянов в системе
- Смены всего и за текущий месяц
- Переключение уведомлений

## 🚀 Установка

1. Клонируйте репозиторий:
   ```bash
   git clone https://github.com/username/schetchik-kalikov.git
   cd "Schetchik Kalиков"
   ```
2. Создайте виртуальное окружение:
   ```bash
   python -m venv venv
   venv\Scripts\activate
   ```
3. Установите зависимости:
   ```bash
   pip install -r requirements.txt
   ```
4. Скопируйте шаблон конфигурации:
   ```bash
   copy .env.example .env
   ```
5. Отредактируйте `.env` и укажите `BOT_TOKEN` и `ADMIN_USER_ID`.

## 📝 Как работать

### Добавление кальяна

1. Нажмите **💼 Смена** → **➕ Добавить кальян**
2. Выберите тип (Стандарт/Сигара)
3. Выберите стол
4. Выберите силу (1–10)
5. Выберите холодность (Холодный/Средний/Теплый)
6. Добавьте комментарий или отправьте `нет`

### Редактирование кальяна (менеджер)

1. **💼 Смена** → **📋 Текущие кальяны**
2. Выберите кальян из списка
3. Выберите действие:
   - ✏️ **Изменить тип**
   - 📍 **Изменить стол**
   - 🔥 **Изменить силу** (1-10)
   - ❄️ **Изменить холодность**
   - 💬 **Изменить комментарий**
   - 🗑️ **Удалить кальян**

### Управление сменой (админ)

1. Откройте или закройте смену через **💼 Смена**
2. В админ-панели (**⚙️ Админ панель** → **📋 Смены**):
   - 📝 **Редактировать время** — изменить время открытия/закрытия (формат: `YYYY-MM-DD HH:MM:SS`)
   - 🔓 **Переоткрыть смену** — переоткрыть закрытую смену
   - 🔒 **Закрыть смену** — закрыть открытую смену
   - 🗑️ **Удалить смену** — удалить смену и все данные

### Назначение глобальной роли (админ)

1. ⚙️ **Админ панель** → **🔧 Роли**
2. Выбрать пользователя
3. Выбрать роль
4. Роль применяется ко ВСЕМ сменам автоматически

## 🚨 Типичные ошибки

- Токен не указан в `.env`
- Неверный `ADMIN_USER_ID`
- Бот не запущен

### Если бот не отвечает

- Проверьте, что `bot.py` запущен
- Убедитесь, что сеть работает
- Перезапустите бота: `Ctrl+C` и снова `python bot.py`
