# 🍃 Счетчик Кальянов - Telegram Bot

Telegram-бот для учета кальянов в кальянной во время рабочей смены. Бот хранит детали заказа, статус кальяна, комментарии и уведомляет команду о новых заявках.

## ✨ Основные возможности

- ✅ Открытие и закрытие смены
- ✅ Добавление кальянов с выбором типа, стола, силы и холодности
- ✅ Комментарии к каждому кальяну
- ✅ Просмотр текущих кальянов и история смен
- ✅ Редактирование типа и стола кальяна
- ✅ Удаление кальянов и админское удаление смены
- ✅ Уведомления для участников смены
- ✅ Автоматическое хранение в SQLite

## 🧩 Роли и доступ

- `admin` — полный доступ к админской панели и удалению смены
- `manager` — редактирование и удаление кальянов
- `hookah_master` — добавление и принятие кальянов
- `supervisor` — просмотр смен и отчетность
- `member` — участие в смене и просмотр данных

## 🚀 Установка

1. Клонируйте репозиторий:
   ```bash
   git clone https://github.com/username/schetchik-kalikov.git
   cd schetchik-kalikov
   ```
2. Создайте виртуальное окружение:
   ```bash
   python -m venv venv
   venv\Scripts\activate
   ```
3. Установите зависимости:
   ```bash
   pip install -r requirements.txt
   ```
4. Скопируйте шаблон конфигурации:
   ```bash
   copy .env.example .env
   ```
5. Отредактируйте `.env` и укажите свой токен и ADMIN_USER_ID.

## ⚙️ Конфигурация

Пример `.env`:

```env
BOT_TOKEN=ваш_токен_здесь
ADMIN_USER_ID=ваш_telegram_user_id
# DB_PATH=hookah_bot.db
# LOG_LEVEL=INFO
```

## 📌 Как пользоваться

### Добавление кальяна

1. Нажмите **Добавить кальян**
2. Выберите тип кальяна
3. Выберите стол
4. Выберите силу (1-10)
5. Выберите холодность
6. Введите комментарий или отправьте `нет`

### Просмотр и редактирование

- Нажмите **Текущие кальяны**
- Выберите кальян из списка
- Измените тип или стол, либо удалите кальян

### Админская панель

- Назначение ролей
- Просмотр расписания
- Удаление текущей смены

## 📊 Первые шаги для админа

- Установите `ADMIN_USER_ID` в `.env`
- Откройте смену
- Добавьте участников при помощи `/assign_role <user_id> <role>`
- Пользователи могут добавлять кальяны и управлять ими в соответствии с ролью

## 🧠 Что сохраняется для каждого кальяна

- Тип
- Стол
- Сила
- Холодность
- Комментарий
- Статус
- Кто добавил
- Последняя правка
- Время создания

## 🐛 Проблемы и решения

- Если бот не стартует — проверьте `.env`
- Если админка не открывается — укажите `ADMIN_USER_ID`
- Для новых полей БД схема обновляется автоматически


### Бот не запускается
```
❌ Токен бота не установлен!
```
**Решение:** Проверьте, что в `.env` файле указан корректный `BOT_TOKEN`

### Ошибка подключения к БД
```
❌ database.sqlite3: no such table
```
**Решение:** Удалите `hookah_bot.db` - БД будет пересоздана автоматически

### Бот не отвечает на команды
**Решение:** 
- Убедитесь, что бот запущен
- Проверьте интернет соединение
- Перезагрузите бот: `Ctrl+C` и снова `python bot.py`
